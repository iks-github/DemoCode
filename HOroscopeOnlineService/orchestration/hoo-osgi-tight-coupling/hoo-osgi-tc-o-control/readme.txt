To run the osgi implementation of HOO do the following:
1. Create in the Eclipse IDE a new OSGi run configuration and modify the first tab (bundles) as follows.
2. Remove all bundles from run config (Press "Select all" with no text entered!)
3. Add the four HOO bundles manually
4. Enter text org.eclipse.osgi and press "Select all"
5. Enter new text org.eclipse.equinox.console and press "Select all"
6. Press "Validate Bundles". No Problems should be detected.
7. Press "Run".
